Below you will find the chronological collection of my projects I have worked on throughout my time at Cal State Dominguez Hills. These projects are being presented as they were completed, without additional time being spent on them outside of the assigned time for the course assignment. These projects were created on an average of two weeks of work alongside other coursework and working fulltime.

While some of these may have touch controls enabled, for best experience I recommend playing these on a computer.
For the 3D projects, a graphics card is required.       



  • First-Person Shooter (Unity)

    HellRazer is a First Person Shooter in which you must annihilate all evil before you. Stick to your guns and fight your way out. This game is Desktop only.

    MAY 2020

  • Third-Person Shooter (Unity)

    Mandalorian: Escape is a Third Person shooter in which you must shoot your way out. You play the role of a Mandalorian securing his next bounty. Things go haywire fast as there are plenty of people trying to stop you. This game is Desktop only.

    MAR 2020

  • 2D Midterm Project

    Dungeons of Hyrule is an action game in which you have to battle your way through a few levels and battle a couple of bosses. This game uses about twice as many events as the two before it and was my midterm project. Uses Keyboard input during gameplay on desktop browsers and mouse to navigate menus. Touch input also available for mobile.

    NOV 2019

  • Top-Down Shooter, Procedural Generation

    Inferno is a Top Down shooting game in which you must collect keys each level to advance to the next. There are more and more zombies every level, and each level is procedurally generated at runtime. Keyboard input on desktop browsing and Touch Input supported for mobile.

    OCT 2019

  • Tower Defense Game

    SURVIVE is a simple turret defense game. You have to place turrets along the walls of the map to defend against the onslaught for 60 seconds. This game uses a mouse input for desktop browsers or touch input on mobile.

    OCT 2019

  • 2D Platformer

    FreeRunner is a simple platform we game in which your goal is to get from point A to point B while avoiding various hazards. This game uses WASD keyboard input or touch controls for mobile.

    SEP
    2019
